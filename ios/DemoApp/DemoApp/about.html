Features:

Speed Test

The Speed Test tab allows the user to compare the performance between their current network and a wide range of cloud and delivery providers.

Instrumented with Cedexis Radar

The app also demonstrates the use of the Cedexis Radar SDK for iOS. Radar allows an iOS app to emit real user monitoring data, as well as real time global network performance data, which is used by Cedexis Openmix to perform rich, logic-based load-balancing. On the Measurements tab, the user can see the data sent by the app.
